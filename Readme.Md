## À propos du projet

Ce projet est mon portfolio personnel, conçu pour mettre en valeur mon parcours en tant que futur ingénieur en cybersécurité et infrastructure réseau. Il présente mes compétences techniques, les projets sur lesquels j'ai travaillé, ainsi que les certifications professionnelles que j'ai obtenues.

![Aperçu du Portfolio](https://user-images.githubusercontent.com/example/screenshot.png)
*(Remplacez le lien ci-dessus par une vraie capture d'écran de votre site)*

### Fonctionnalités

-   **Design Moderne et Responsive :** Entièrement consultable sur mobile et ordinateur.
-   **Animations :** Transitions fluides au défilement grâce à la bibliothèque AOS.
-   **Sections Complètes :**
    -   **Accueil :** Une introduction percutante.
    -   **À Propos :** Mon parcours et mes ambitions.
    -   **Compétences :** Un aperçu de mes compétences techniques.
    -   **Projets :** Une galerie de mes réalisations avec des liens vers les démos et le code source.
    -   **Certifications :** Mes certifications vérifiables en ligne.
    -   **Services :** Les prestations que je propose.
    -   **Contact :** Un formulaire pour me joindre facilement.

### Technologies utilisées

-   **Frontend:**
    -   HTML5
    -   CSS3
    -   [Tailwind CSS](https://tailwindcss.com/)
    -   JavaScript (ES6+)
-   **Bibliothèques:**
    -   [Font Awesome](https://fontawesome.com/) (pour les icônes)
    -   [Animate On Scroll (AOS)](https://michalsnik.github.io/aos/) (pour les animations)
-   **Outils de développement:**
    -   [Node.js](https://nodejs.org/) & [npm](https://www.npmjs.com/)
    -   [PostCSS](https://postcss.org/) & [Autoprefixer](https://github.com/postcss/autoprefixer)

## Démarrage

Suivez ces étapes pour obtenir une copie locale du projet et la faire fonctionner.

### Prérequis

Assurez-vous d'avoir Node.js et npm installés sur votre machine.
- **npm**
  ```sh
  npm install npm@latest -g
  ```

### Installation

1.  Clonez le dépôt GitHub :
    ```sh
    git clone https://github.com/votre-utilisateur/votre-repo.git
    ```
    *(Remplacez l'URL par celle de votre dépôt)*
2.  Accédez au dossier du projet :
    ```sh
    cd votre-repo
    ```
3.  Installez les dépendances npm :
    ```sh
    npm install
    ```
4.  Lancez le script de build pour compiler le CSS avec Tailwind :
    ```sh
    npm run build:css
    ```
    Ou utilisez le mode "watch" pour recompiler automatiquement lors des modifications :
    ```sh
    npm run build:css:watch
    ```
5.  Ouvrez le fichier `index.html` dans votre navigateur pour voir le site.
